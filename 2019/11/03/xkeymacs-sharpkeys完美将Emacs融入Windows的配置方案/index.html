xkeymacs + sharpkeys完美将Emacs融入Windows的配置方案
Post：2019-11-03 21:21:21 Tags：/ emacs / sharpkeys / xkeymacs / Visit：

xkeymacs

来源

  • GitHub开源项目
  • 传送门

作用

  • 几乎完美将Emacs融入Windows，在Windows上使用Emacs快捷键

下载地址

  • 传送门

注意事项

  • Ctrl + Q 为暂时开关Emacs快捷键
  • 在Chrome浏览器无法使用CTRL + F （如果要使用，先关闭ctrl + Q）
  • 也是Chrome浏览器，Ctrl + W 等快捷键均无法使用，建议安装vimium使用（这个插件非常好用！）

sharpkeys

来源

  • GitHub开源项目
  • 传送门

作用

  • 这个软件是通过修改注册表，将键位修改，常见emacser修改caps键与CTRL键互换，vimer修改esc与caps键互换。
  • 可视化傻瓜操作，无敌好用（不过我之前都是直接代码修改注册表的。。。）
